# Patterns — overview

Каталог проверенных проектировочных паттернов для AI-агентов. Вместо изобретения велосипеда — используем готовые шаблоны взаимодействия агента с окружением, другими агентами и пользователем.

## Материалы

- [Skills и правила для агентов](agent-skills-and-rules.md) — модульные инструкции, проектные правила и проверяемые workflow
- [Антипаттерны агентных систем](agent-antipatterns.md) — over-agentification, blind tool execution, weak evals и hidden costs

## Как устроен раздел

Каждый паттерн описывается по схеме: проблема → решение → псевдокод/диаграмма → когда применять → связанные паттерны. Паттерны активно переиспользуются в кейсах и архитектуре.
