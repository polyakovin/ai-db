# Паттерны работы с AI-агентами

Концепции и архитектурные паттерны для эффективной работы с AI-агентами.

## Компоненты системы

### Harness
Оркестратор агента — управляет жизненным циклом, контекстом и выполнением задач.

Подробнее: [Agent Harness](../patterns/agent-harness.md).

### Ретроспективная обработка ошибок ИИ
Механизм анализа и обработки ошибок, укрепляющий харнесс через обратную связь.

### System Prompt
Базовая конфигурация поведения агента — роль, ограничения, принципы работы.

### AGENTS.md
Файл конфигурации проекта для агентов — описание проекта, команды, процессы.

### Skills
Навыки агента — подгружаются по требованию для конкретных задач.

Подробнее: [Skills и правила для агентов](../patterns/agent-skills-and-rules.md).

### Commands
Команды для управления агентом (опционально).

### Hooks
Автоматизация реакций на действия агента:
- CI/CD пайплайны
- Тесты и линтеры
- Валидация изменений

### Agents (Souls) / Subagents
Делегирование специфических задач отдельным агентам с изолированным контекстом.

### MCP (Model Context Protocol)
«Руки» агента — возможность управлять существующими инструментами и сервисами.

### Integrations
Внешние интеграции (опционально).

### ACP
Протокол взаимодействия агентов.

### Channels
Каналы коммуникации между агентами и компонентами.

### Общение с CLI
Интерфейс для взаимодействия через командную строку.

### Tools
Инструменты для выполнения задач.

### Cron / Heartbeat
Планировщик задач — расписание и периодические проверки.
