# OpenAI Academy — бесплатное обучение AI-агентам

**Источник:** https://academy.openai.com/  
**Формат:** Бесплатные курсы, livestream-ивенты, on-demand записи, блог

## Структура курсов

Три уровня сложности:

### AI Foundations — для начинающих
- Основы AI, LLM и ChatGPT
- Как давать чёткие инструкции, добавлять контекст, проверять выводы
- Ответственное использование AI в повседневной работе

### Applied AI Foundations — для пользователей с опытом
- Превращение повторяющихся задач в повторяемые workflow
- Декомпозиция работы на шаги, поиск мест где ChatGPT помогает
- Встраивание точек проверки в процесс

### Agents and Workflows — для уверенных пользователей
- Управление агентами через структурированную работу
- Контекст, определение вывода, границы, ревью черновиков
- Улучшение и переиспользование workflow

## Регулярные ивенты

| Ивент | Описание |
|---|---|
| **Builder Bootcamp: Agents** | Раз в месяц — сборка агентов |
| **Builder Bootcamp: Codex** | Раз в месяц — работа с Codex |
| **Builder Bootcamp: Evals** | Оценка качества |
| **Builder Bootcamp: RAG** | Retrieval-Augmented Generation |
| **How X teams use Codex** | Отраслевые кейсы: маркетинг, продажи, data science, бизнес-операции |

## Что ещё есть
- **Higher Education Club** — вебинары для преподавателей и исследователей
- **Work Users Club** — кейсы использования Codex в разных отделах
- **Builders Club** — технические воркшопы для разработчиков
- **Блог** — практические статьи (закупки, госсектор, малый бизнес)
