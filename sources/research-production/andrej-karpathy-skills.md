# Andrej Karpathy Skills (MultiCAI)

**URL:** https://github.com/multica-ai/andrej-karpathy-skills

Навыки и промпты Андрея Карпати для AI-агентов — коллекция практик и паттернов работы с LLM.

## Описание

Набор навыков (skills) для эффективного использования AI-моделей в разработке и исследовании.

## Обзор ресурса

- **Автор:** MultiCAI (multica-ai) — сообщество/организация, систематизирующая best practices для AI coding agents
- **Лицензия:** Open-source (см. репозиторий)
- **Статус:** Вирусный репозиторий (2026) — стал популярным после публикации наблюдений Андрея Карпати об ошибках LLM при кодинге
- **Содержание:** Один `CLAUDE.md` файл + плагин для Claude Code + правило для Cursor — 4 принципа: минимализм кода, хирургические изменения, явные предположения, проверяемые критерии успеха
- **Установка:** `/plugin install andrej-karpathy-skills@karpathy-skills` в Claude Code
- **Философия:** «Не копируй слепо — адаптируй под свой репозиторий»: используй как меню правил, не как шаблон
- **Relevance:** Ключевой референс для понимания того, какие «рельсы» нужны coding agents для предотвращения типичных ошибок (overcomplication, hallucination, side effects)
