# Perplexity AI

AI-поисковая платформа с агентными возможностями («ответная машина»). Позиционируется как быстрый и точный поиск с цитированием источников — и активно эволюционирует в сторону AI-агента.

**Computer** — агент для multi-step задач (исследования, написание, анализ). Spaces, Workflows, Skills, Connectors, Artifacts — всё это формирует полноценную агентную обвязку.

## Релевантность

Perplexity — показательный пример эволюции классического поискового движка в consumer-grade AI-агента с multi-step execution, внешними инструментами и персистентным контекстом. Полезен как reference для проектирования пользовательского агентного интерфейса.
