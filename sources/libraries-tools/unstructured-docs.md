# Unstructured Docs

Официальная документация Unstructured по ingestion и pre-processing неструктурированных документов.

## Описание

- **Основные темы:** partitioning, cleaning, extracting, chunking, document metadata, source/destination connectors, supported file types.
- **Практическая ценность:** reference для превращения PDF, HTML, Word, изображений и других файлов в пригодные для RAG элементы.
- **Relevance:** useful reference для document parsing/OCR слоя automated context collection.
