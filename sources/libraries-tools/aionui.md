## Что это

Desktop-приложение для multi-agent coworking: агенты живут на твоём компьютере, работают параллельно, общаются между собой и управляются удалённо. Free & open-source.

## Ключевые особенности

### Multi-Agent Cowork
- Авто-детектит Claude Code, Codex, Gemini CLI и 20+ других инструментов уже установленных на машине
- Запуск агентов параллельно, распределение задач между ними
- Единый workspace с live activity feed

### 20+ Built-in Assistants
Специализированные агенты без настройки: Slide Creator, Excel Creator, Dashboard (data viz), UI/UX Designer, 3D Game Creator, Academic Paper, Life Coach, Pitch Deck, Diagram Expert и другие. Можно создавать своих.

### Remote Control
- Управление агентами с телефона через Telegram, WeChat, Lark, DingTalk, WebUI
- Агенты продолжают работать на компьютере

### 24/7 Automation
- Cron-расписание для агентов: код-ревью каждое утро, бэкап каждый вечер, отчёт каждый понедельник

### Platform
- **Любая LLM** — разные модели разным агентам, прямые API-вызовы
- **Skills Marketplace** — community skills, можно создавать и делиться
- **Unified MCP** — один раз подключить MCP сервер, синхронизируется со всеми агентами
- **Office Toolkit** — генерация PPT, Excel, Word, PDF нативно, без облака
