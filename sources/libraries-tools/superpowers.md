# Superpowers

**URL:** https://github.com/obra/Superpowers

Open-source проект для усиления возможностей AI-агентов и автоматизации рабочих процессов.

## Описание

Инструменты и паттерны для расширения функциональности AI-ассистентов.

## Обзор ресурса

- **Автор:** Jesse Vincent (obra) / Prime Radiant
- **Лицензия:** MIT
- **Звёзды:** ~239K на GitHub (июнь 2026) — один из самых популярных AI-coding инструментов
- **Релизы:** 8 релизов; последний — v6.0.3 (18 июня 2026)
- **Контрибьюторы:** 43+
- **Поддерживаемые платформы:** Claude Code, Codex (App и CLI), Cursor, Gemini CLI, GitHub Copilot CLI, Kimi Code, OpenCode, Pi, Factory Droid
- **Философия:** Spec-driven development — агент сначала проектирует (brainstorm → plan → micro-tasks), потом исполняет через sub-agents, с TDD и code review. Борется с «vibe coding»
- **Relevance:** Де-факто стандарт agentic skills framework для AI coding agents
