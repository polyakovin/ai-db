# AI Foundations — YouTube-канал

**Ссылка на канал:** https://www.youtube.com/@AI-Foundations
**Подписчики:** ~338K
**Тематика:** Claude, AI-агенты, автоматизация, Cowork, Codex

## О канале

Канал AI Foundations — один из крупнейших ресурсов по Claude на YouTube. Производят полные гайды «от новичка до PRO» по всей экосистеме Claude. Сквозные, фундаментальные, структурированные — идеально для базы знаний.

Авторы также ведут платформу [aifoundations.io](https://aifoundations.io) с платными курсами.

## Типичный контент

- Сквозные туториалы «для начинающих до PRO»
- Пошаговые гайды по Claude Cowork и Claude Code
- Сборка Skills с нуля
- Деплой реальных проектов (Netlify, GitHub)
- Сравнение Claude Web vs Claude Cowork vs Claude Code

## Обзор ресурса

- **Автор/владелец:** AI Foundations (aifoundations.io) — одна из крупнейших образовательных платформ по Claude
- **Подписчики:** ~341K (по состоянию на июнь 2026)
- **Контент:** 160+ видео; основные плейлисты — Claude Tutorials, Claude Cowork, Claude Code
- **Модель монетизации:** Бесплатный YouTube-контент + платные курсы на aifoundations.io и skool.com
- **Актуальность:** Высокая — регулярные обновления (последние видео — июнь 2026), покрывают все новые фичи Claude
- **Уникальность:** Сквозные гайды «от новичка до PRO», покрывающие всю экосистему Claude целиком
