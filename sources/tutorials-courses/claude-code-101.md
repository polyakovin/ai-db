- **Платформа:** Anthropic Skilljar (официальная образовательная платформа Anthropic)
- **Формат:** Бесплатный онлайн-курс с интерактивными модулями, отслеживанием прогресса и сертификатом
- **Дата запуска:** ~2025-2026 (в рамках Anthropic Academy)

- **Контекст:** Входит в экосистему Anthropic Academy наряду с курсами по API, MCP, Cowork. Охватывает prompting, Projects, Artifacts, Skills, Connectors, Research Mode, Claude Code и многое другое (~3+ часа контента)
