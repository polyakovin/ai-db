# Add Note — добавление заметки в базу знаний

## Когда использовать
Нужно добавить новую концепцию, паттерн или заметку в ai-db.

## Алгоритм
1. Определи тип материала (concept / pattern / tool / practice / case / source).
2. Создай файл в соответствующем разделе (kebab-case, английское название).
3. Для source — заполни frontmatter (title, url, type, category, tags, added, status).
4. Добавь ссылку в `overview.md` раздела.
5. Создай ссылки на связанные заметки.
6. Запусти `bash meta/scripts/validate-vault.sh`.

## Проверка
- Файл создан в правильном разделе.
- Имя в kebab-case, на английском.
- overview.md раздела обновлён.
- validate-vault.sh проходит без ошибок.
- Есть обратные ссылки из связанных заметок (если нужно).
