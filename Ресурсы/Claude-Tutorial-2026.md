# Claude Tutorial — от новичка до PRO (2026)

**Источник:** YouTube, AI Foundations  
**Ссылка:** https://youtu.be/rRrBbyv3ChM  
**Формат:** Полный гайд по Claude (1ч 52мин) — от регистрации до Claude Code и деплоя

## О чём видео

Сквозной туториал по всей экосистеме Claude: начало с нуля → продвинутые фичи → Claude Code → продакшн.

## Структура

### Основы
- Регистрация, цены, настройки, интерфейс
- Claude Memory, экспорт из ChatGPT
- Модели, Writing Styles, Voice, Keyboard Shortcuts
- Организация чатов, Incognito, GCAO Prompt Template

### Claude Projects
- Project Instructions, Artifacts Library
- Создание и шаринг артефактов
- Inline-визуализации, анализ данных

### Claude Desktop / Cowork
- Computer Use, Scheduled Tasks
- Connectors: Notion (MCP), Gmail, Google Calendar
- Сборка Claude Skills (DBS Framework)

### Claude Code
- Два режима: локальный и Remote Environment (на серверах Anthropic)
- Plan Mode, CLAUDE.md, Effort настройки
- Деплой сайта на Netlify, свой домен
- GitHub + Netlify CI/CD
- Scheduled Tasks с Skills
- Управление из телефона

## Ключевые моменты

- **Claude.md** — аналог AGENTS.md для проекта, базовый файл инструкций
- **Plan Mode** — сначала план, потом код
- **Connectors** — MCP-подключения к внешним сервисам
- **Skills** — кастомные скиллы, которые можно переиспользовать
- **Computer Use** — Claude управляет компьютером напрямую
- **Remote Environment** — запуск Claude Code на серверах Anthropic, доступ с любого устройства
