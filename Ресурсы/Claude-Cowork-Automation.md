Практический гайд по Claude Cowork: как настроить автоматизацию соцсетей через Skills + Connectors. Весь пайплайн: research → создание контента → публикация.

## Структура

### 1. Настройка workspace (01:14)
- Организация рабочего пространства Cowork
- Базовая конфигурация перед запуском

### 2. Skills и Connectors (04:06)
- Подключение внешних сервисов через коннекторы
- Сборка Skills для повторяющихся задач

### 3. Связанный workflow (07:52)
- Сквозной пайплайн: research → контент → публикация
- Blotato как инструмент для постинга

### 4. Полная автономия (14:38)
- Scheduled Tasks — выполнение без участия человека
- Замкнутый цикл автоматизации

## Ключевые моменты

- **Blotato** — сервис для публикации контента, интеграция с Cowork через коннектор
- **Skills** — переиспользуемые блоки: research skill, content skill, publish skill
- **Scheduled Tasks** — Cowork может работать по расписанию полностью автономно
- Реальный кейс: полное управление соцсетями через Claude Cowork
