# Сравнение платформ — Обзор

Сравнительный анализ платформ и экосистем для создания AI-агентов. Задача раздела — дать трезвую картину рынка: кто на что способен, где сильные и слабые стороны, что выбрать под конкретный сценарий.

## Что входит в раздел

- [LangChain vs Semantic Kernel vs AutoGen](LangChain%20vs%20Semantic%20Kernel%20vs%20AutoGen.md) — сравнение трёх главных фреймворков
- [Dify vs Flowise vs LangFlow](Dify%20vs%20Flowise%20vs%20LangFlow.md) — low-code платформы: что выбрать для прототипирования
- [OpenAI Assistants vs Anthropic Claude Agent](OpenAI%20Assistants%20vs%20Anthropic%20Claude%20Agent.md) — сравнение нативных агентных API вендоров
- [Open-source vs Managed](Open-source%20vs%20Managed.md) — самостоятельное развёртывание против облачных сервисов
- [Сравнение LLM для агентов](Сравнение%20LLM%20для%20агентов.md) — бенчмарки function calling, планирования, следования инструкциям
- [Векторные базы для RAG](Векторные%20базы%20для%20RAG.md) — Pinecone vs Weaviate vs Qdrant vs pgvector: цена и скорость
- [Чек-лист выбора](Чек-лист%20выбора.md) — готовая матрица: что брать под конкретный use-case

## Как устроен раздел

Каждая заметка — это сравнительная таблица с критериями, плюсами, минусами и рекомендациями. Активно ссылается на [Инструментарий](Инструментарий/Обзор.md) и [практические кейсы](Кейсы-практики/Обзор.md).
