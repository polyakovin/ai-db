# Сравнение платформ — Обзор

Сравнительный анализ платформ и экосистем для создания AI-агентов. Задача раздела — дать трезвую картину рынка: кто на что способен, где сильные и слабые стороны, что выбрать под конкретный сценарий.

## Что входит в раздел

- LangChain vs Semantic Kernel vs AutoGen — сравнение трёх главных фреймворков
- Dify vs Flowise vs LangFlow — low-code платформы: что выбрать для прототипирования
- OpenAI Assistants vs Anthropic Claude Agent — сравнение нативных агентных API вендоров
- Open-source vs Managed — самостоятельное развёртывание против облачных сервисов
- Сравнение LLM для агентов — бенчмарки function calling, планирования, следования инструкциям
- Векторные базы для RAG — Pinecone vs Weaviate vs Qdrant vs pgvector: цена и скорость
- Чек-лист выбора — готовая матрица: что брать под конкретный use-case

## Как устроен раздел

Каждая заметка — это сравнительная таблица с критериями, плюсами, минусами и рекомендациями. Активно ссылается на Инструментарий и практические кейсы.
