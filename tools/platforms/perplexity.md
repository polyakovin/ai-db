# Perplexity AI

AI-поисковая платформа с агентными возможностями. Позиционируется как «ответная машина» — даёт быстрые и точные ответы из проверенных источников.

## Агентные возможности

Perplexity активно развивает функциональность в сторону AI-агентов:

| Возможность | Описание |
|-------------|----------|
| **Computer** | Агент, выполняющий multi-step задачи: исследования, написание, анализ — «hand off your projects to get polished, reliable deliverables around the clock» |
| **Spaces** | Тематические рабочие пространства с собственным контекстом и знаниями |
| **Workflows** | Многошаговые автоматизированные пайплайны обработки запросов |
| **Skills** | Расширяемые возможности агента (подключаемые модули) |
| **Connectors** | Интеграция с внешними источниками (Google Drive, Slack, etc.) |
| **Artifacts** | Сохранение результатов работы агента |

## Отраслевые режимы

- **Discover** — общий поиск
- **Finance** — финансовые данные и аналитика
- **Health** — медицинская информация
- **Academic** — академические источники
- **Patents** — патентный поиск

## Релевантность для базы знаний

Perplexity — пример эволюции поискового движка в сторону AI-агента с multi-step execution, внешними инструментами и персистентным контекстом (Spaces). Полезен как reference-реализация consumer-grade AI-агента.
