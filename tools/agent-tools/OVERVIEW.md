# Agent Tools — обзор

Инструменты, которые AI-агенты используют для выполнения задач: веб-поиск, работа с файлами, выполнение кода, браузерная автоматизация, API-клиенты и другие утилиты.

## Категории инструментов

### 🔍 Поиск и исследование

- [Web Search](web-search.md) — поисковые инструменты для агентов (Tavily, Exa, Serper)
- [Browser Automation](browser-automation.md) — управление браузером (Playwright, Puppeteer, Browser-use)

### 📁 Файлы и данные

- [File System](file-system.md) — чтение/запись файлов, навигация по директориям

### 💻 Код и выполнение

- [Code Execution](code-execution.md) — запуск кода в песочнице (E2B, Jupyter, Docker)

### 🔌 Интеграции

- [API Clients](api-clients.md) — HTTP-клиенты для внешних сервисов

### 🧰 Универсальные

- [Function Calling](function-calling.md) — механизм вызова функций через LLM

## Как устроен раздел

Каждая заметка описывает конкретный тип инструментов или библиотеку:

- **Назначение** — какую задачу решает
- **Ключевые возможности** — что умеет
- **Примеры использования** — типичные сценарии
- **Интеграция с агентами** — как подключается к harness
- **Безопасность** — sandbox, ограничения, риски

### Single Source of Truth

Каждый инструмент имеет **единственную canonical страницу** в `tools/agent-tools/` или смежных разделах (`tools/frameworks/`, `tools/platforms/`). Все паттерны и источники **только ссылаются** на эти страницы.
