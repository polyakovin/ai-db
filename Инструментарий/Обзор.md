# Инструментарий — Обзор

Раздел охватывает фреймворки, библиотеки, SDK и утилиты для создания, отладки и развёртывания AI-агентов. Цель — дать практический обзор доступного инструментария и помочь с выбором под конкретную задачу.

## Что входит в раздел

- Фреймворки для агентов — LangChain, Semantic Kernel, CrewAI, AutoGen, Dify: сравнение возможностей
- Инструменты и плагины — MCP-серверы, Function Calling, сторонние API и интеграции
- Среды исполнения — песочницы, Docker-образы, serverless-рантаймы для агентов
- Векторные базы данных — Pinecone, Chroma, Qdrant, Milvus: выбор под RAG и память
- Мониторинг и трассировка — LangSmith, Phoenix, OpenTelemetry: как отлаживать агентные цепочки
- UI и визуальные конструкторы — Dify, Flowise, LangFlow: low-code сборка агентов
- Тестирование и бенчмарки — SWE-bench, GAIA, AgentBench: как измерять качество агентов
- Деплой и продакшн — оркестрация, масштабирование, CI/CD для агентных систем

## Как устроен раздел

Каждая заметка — это обзор конкретного инструмента или категории инструментов, с примерами кода, таблицами сравнения и рекомендациями по выбору. Перекрёстные ссылки ведут к паттернам и практическим кейсам.
