# Recipes — Обзор

Коллекция готовых рецептов: пошаговых инструкций для решения конкретных задач с AI-агентами. Минимум теории — максимум воспроизводимого кода и конфигураций.

## Что входит в раздел

- [RAG-агент с нуля](RAG-агент с нуля.md) — пошаговая сборка retrieval-augmented агента: от индексации до ответа
- [Агент для SQL-запросов](Агент для SQL-запросов.md) — text-to-SQL агент с песочницей и защитой от инъекций
- [Многоагентный воркфлоу](Многоагентный воркфлоу.md) — CrewAI/AutoGen: сборка команды агентов для исследования
- [Агент с долгосрочной памятью](Агент с долгосрочной памятью.md) — подключение векторной БД и суммаризации для persistent-памяти
- [Telegram](Агент в Slack/Telegram.md) — интеграция агента в мессенджер: вебхуки, асинхронность, безопасность
- [Function Calling с fallback](Function Calling с fallback.md) — надёжная работа с инструментами: retry, валидация, degradation
- [Self-Correcting Agent](Self-Correcting Agent.md) — агент, который проверяет и исправляет свои ответы

## Как устроен раздел

Каждый рецепт — самодостаточный документ с кодом, конфигурациями и ожидаемым результатом. Рецепты опираются на [паттерны](Паттерны/Обзор.md) и [инструменты](Инструментарий/Обзор.md), давая их практическое применение.
