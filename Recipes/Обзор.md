# Recipes — Обзор

Коллекция готовых рецептов: пошаговых инструкций для решения конкретных задач с AI-агентами. Минимум теории — максимум воспроизводимого кода и конфигураций.

## Что входит в раздел

- RAG-агент с нуля — пошаговая сборка retrieval-augmented агента: от индексации до ответа
- Агент для SQL-запросов — text-to-SQL агент с песочницей и защитой от инъекций
- Многоагентный воркфлоу — CrewAI/AutoGen: сборка команды агентов для исследования
- Агент с долгосрочной памятью — подключение векторной БД и суммаризации для persistent-памяти
- Telegram — интеграция агента в мессенджер: вебхуки, асинхронность, безопасность
- Function Calling с fallback — надёжная работа с инструментами: retry, валидация, degradation
- Self-Correcting Agent — агент, который проверяет и исправляет свои ответы

## Как устроен раздел

Каждый рецепт — самодостаточный документ с кодом, конфигурациями и ожидаемым результатом. Рецепты опираются на паттерны и инструменты, давая их практическое применение.
