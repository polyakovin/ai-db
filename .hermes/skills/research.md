# Research — исследование и добавление источников

## Когда использовать
Нужно найти информацию по теме, изучить внешний источник и добавить его в базу знаний.

## Алгоритм
1. Используй web_search / web_extract для поиска материалов.
2. Отдели факты от выводов.
3. Зафиксируй дату актуальности.
4. Создай заметку в `sources/` по шаблону `meta/templates/source.md`.
5. Если источник существенный — создай концепцию/паттерн в соответствующем разделе.
6. Добавь ссылки из overview-страниц.
7. Запусти `bash scripts/validate-vault.sh`.

## Проверка
- Источник добавлен в `sources/` с полным frontmatter.
- Дата актуальности зафиксирована.
- validate-vault.sh проходит.
- Есть ссылки из смежных разделов.
