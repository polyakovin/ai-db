# Паттерны — Обзор

Каталог проверенных проектировочных паттернов для AI-агентов. Вместо изобретения велосипеда — используем готовые шаблоны взаимодействия агента с окружением, другими агентами и пользователем.

## Что входит в раздел

- [[ReAct]] — Reasoning + Acting: базовый паттерн «подумал → сделал → посмотрел»
- [[Chain-of-Thought]] — явные рассуждения перед действием, включая многошаговые варианты
- [[Декомпозиция и делегирование]] — Plan-and-Solve, divide-and-conquer для агентов
- [[Рефлексия и самооценка]] — Reflection, Self-Refine, Critic-агенты: проверка собственных результатов
- [[Инструментальные паттерны]] — грамотный function calling, fallback-стратегии, retry-логика
- [[Многоагентные паттерны]] — дебаты, консенсус, Mixture-of-Agents, ранжирование ответов
- [[Управление диалогом]] — clarification, confirmation, summarization: как агент общается с человеком
- [[Кэширование и мемоизация]] — когда агент должен запоминать предыдущие результаты

## Как устроен раздел

Каждый паттерн описан по схеме: проблема → решение → псевдокод/диаграмма → когда применять → связанные паттерны. Паттерны активно переиспользуются в [[Кейсы-практики/Обзор|кейсах]] и [[Архитектура-агентов/Обзор|архитектуре]].
