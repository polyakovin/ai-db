# Паттерны — Обзор

Каталог проверенных проектировочных паттернов для AI-агентов. Вместо изобретения велосипеда — используем готовые шаблоны взаимодействия агента с окружением, другими агентами и пользователем.

## Что входит в раздел

- [ReAct](ReAct.md) — Reasoning + Acting: базовый паттерн «подумал → сделал → посмотрел»
- [Chain-of-Thought](Chain-of-Thought.md) — явные рассуждения перед действием, включая многошаговые варианты
- [Декомпозиция и делегирование](Декомпозиция и делегирование.md) — Plan-and-Solve, divide-and-conquer для агентов
- [Рефлексия и самооценка](Рефлексия и самооценка.md) — Reflection, Self-Refine, Critic-агенты: проверка собственных результатов
- [Инструментальные паттерны](Инструментальные паттерны.md) — грамотный function calling, fallback-стратегии, retry-логика
- [Многоагентные паттерны](Многоагентные паттерны.md) — дебаты, консенсус, Mixture-of-Agents, ранжирование ответов
- [Управление диалогом](Управление диалогом.md) — clarification, confirmation, summarization: как агент общается с человеком
- [Кэширование и мемоизация](Кэширование и мемоизация.md) — когда агент должен запоминать предыдущие результаты

## Как устроен раздел

Каждый паттерн описан по схеме: проблема → решение → псевдокод/диаграмма → когда применять → связанные паттерны. Паттерны активно переиспользуются в [кейсах](Кейсы-практики/Обзор.md) и [архитектуре](Архитектура-агентов/Обзор.md).
