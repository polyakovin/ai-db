При коммите и пуше в защищённую ветку — создавать MR автоматически, без уточняющих вопросов.

**Why:** Пользователь явно указал «не спрашивай больше» — предпочитает автоматическое создание MR вместо диалога.

**How to apply:** После успешного коммита, если пуш отклонён из-за защищённой ветки, сразу создавать MR через glab без подтверждения.
