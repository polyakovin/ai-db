---
name: правила хранятся внутри проекта
description: Память/правила сохраняются в .nessy/memory/ внутри проекта, а не в глобальной директории
type: feedback
---

Все файлы памяти (feedback, project, user, reference) сохранять в `.nessy/memory/` внутри текущего проекта, а не в глобальной директории `~/.nessy/projects/.../memory`.

**Why:** Пользователь явно указал хранить правила «на уровне проекта» — для локальности и переносимости конфигурации.

**How to apply:** При сохранении новых правил создавать файлы в `.nessy/memory/<project>/` и обновлять `.nessy/memory/MEMORY.md` внутри проекта.
